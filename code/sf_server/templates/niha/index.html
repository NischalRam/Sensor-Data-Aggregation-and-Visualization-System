Welcome to Our Project Sensor Data Aggregation and Visualization System

Project Overview

The goal of this project is to develop an extensive software system that collects data from a network of sensors, combines the information gathered, and delivers it to end users via a user-friendly graphical user interface (GUI) hosted on a server.
